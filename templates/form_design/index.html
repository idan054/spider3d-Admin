ניהול אפליקציית Spider3D

*יש לשים לב היטב לאופן השימוש למניעת תקלות*

- מוצר/ קטגוריה: יש להקליד product / category כדי לבחור לאן תוביל התמונה.
- מס׳ קבוצה: יש לסמן ב 1 (ברירת מחדל) את כל הסליידרים באותה הקבוצה.
- סדר הרשימה הוא סדר הצוגה כך שהסליידר הראשון מוצג ראשון...
- מס׳ מוצר/ קטגוריה: יש ולהעתיק את ה id שלו מהקישור בלעמוד העריכה של המוצר המתאים (לדוגמא tag_ID=2352).
- קישור לתמונה: קישור ישיר לתמונה, יחס מומלץ 18:9 (938×624), התמונה נחתכת מעט מלמטה ולמעלה.
הערה: כדי להרים את שורת הקטגוריות לראש, יש לבחור במס׳ קבוצה 2.
דוגמא למילוי הפאנל ניהול.


To update the whole Config file push main branch on 'Spider3DFlux' Repo.
To update The sliders just click the "עדכן" Button. (This will use the live config at https://config-fluxstore-mauve.vercel.app to update the sliders)


This Cpanel run on Heroku server (at https://spider3d-admin.herokuapp.com), who adapted to flask deploys.
Push 'spider3d-admin' repo will auto update this Cpanel on the server, Logs available on idanbit80@gmail.com user in Heroku site under Activity
